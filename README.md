Power BI – Lucro, Vendas e Unidades Vendidas

📌 **Descrição do Projeto**
Este repositório faz parte da minha jornada de desafios com Power BI na DIO.
O objetivo foi replicar páginas já criadas durante o curso e desenvolver uma nova página original, explorando diferentes visuais e consolidando habilidades de criação de relatórios interativos.

📊 **Estrutura do Relatório**
O relatório final é composto por 3 páginas:

**Página 1 e Página 2**
Replicadas conforme o curso, utilizando a sample disponibilizada.
Contêm visuais de análise de vendas e desempenho.

**Página 3** 
**Mapa 1:** Vendas e Unidades Vendidas por País
**Mapa 2:** Lucro por País
**Gráfico de Pizza:** Lucro por Segmento
**Treemap:** Participação de Lucro por Segmento

👉 Todos os visuais foram renomeados para títulos claros e diretos, com tooltips configurados para enriquecer a experiência do usuário.

🛠️ **Ferramentas Utilizadas**
Power BI Desktop
Power BI Service (para publicação)
GitHub (versionamento e portfólio)



📂 **Estrutura do Repositório**

Relatorio.pbix → Arquivo do Power BI.
README.md → Documentação do projeto.


🎯 **Objetivos do Desafio**

Reforçar conceitos de criação de visuais no Power BI.
Exercitar boas práticas de layout e organização.
Publicar e compartilhar relatórios interativos.
Documentar o projeto para compor portfólio no GitHub.

📈 **Resultados**

Relatório publicado com 3 páginas funcionais.
Visuais claros, bem organizados e em português.
Projeto versionado e disponível para consulta.
